# UART
 - 시간을 기준으로 데이터 송,수신(baudrate)
 - 범용 비동기 송수신기
 - 속도를 올리는대 제한이 있음
 - 1:1 통신
 - Start Bit 일때는 Low, Stop Bit 일때는 High

 # I2C
  - CLK을 기준으로 데이터 송.수신
  - 1:N 통신
  - 그냥 불러서 통신

 # SPI
  - CLK을 기준으로 데이터 송.수신
  - 속도가 제일 빠른 장치
  - 1:N 통신
  - 선택한거만 데이터 송.수신
